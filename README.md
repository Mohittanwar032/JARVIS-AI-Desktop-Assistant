# 🤖 JARVIS AI Desktop Assistant

An AI-powered voice-controlled desktop assistant inspired by JARVIS, built using Python, Flask, JavaScript, and the Gemini API.

JARVIS combines conversational AI with local desktop automation, allowing users to interact with their computer using natural voice commands.

## 🚀 Features

- 🎙️ Voice command recognition
- 🔄 Continuous listening mode
- 🔊 Text-to-speech responses
- 🧠 Gemini-powered AI conversations
- 💻 Local desktop automation
- 🌐 Open websites using voice commands
- 🎵 Search and play content on YouTube
- 📷 Open and close the Windows Camera
- 🧮 Launch Calculator
- 📝 Launch Notepad
- 🎨 Launch Paint
- 💻 Open Command Prompt
- 🔎 Google voice search
- 🖥️ Futuristic JARVIS-inspired HUD
- ⚡ Local command routing to reduce unnecessary AI API calls
